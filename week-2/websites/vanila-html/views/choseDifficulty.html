Hangman Game Tutorial

Objective:Guess the hidden word one letter at a time before the hangman is fully drawn.

How to play

  1. Start:The game selects a secret word, and you see blanks (_) representing each letter.
  2. Guess Letters: Enter one letter at a time.
    • Correct Guess: The letter fills in the blanks where it appears.
    • Wrong Guess: A part of the hangman drawing is added.
  3. Win: Guess all letters before the drawing is complete.
  4. Lose: If the hangman is fully drawn before you guess the word.
